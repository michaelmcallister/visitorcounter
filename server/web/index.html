Usage

Simply include an image tag linking to "c.png" on this domain and set the 'd' parameter to your domain. There's no need to register or signup. No emails, no credit cards. 

<img src="https://hit.sknk.ws/c.png?d=example.com" />

Themes

To change themes simply pass in a query parameter 't' with a number that corresponds to the theme below. For example to select theme two you would add the image to your website like so: 

<img src="https://hit.sknk.ws/c.png?t=2&d=example.com" />
By default the theme is 1.
