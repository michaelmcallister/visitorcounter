Usage

Simply include an image tag linking to "c.png" on this domain. There's no need to "register" or signup. No emails, no credit cards. 

<img src="https://hit.sknk.ws/c.png" />

Themes

To change themes simply pass in a query parameter 't' with a number that corresponds to the theme below. For example to select theme two you would add the image to your website like so: 

<img src="https://hit.sknk.ws/c.png?t=2" />
By default the theme is 1.

How it works

The server looks at the Referer header of the request to determine the domain. It's also possible to override this (in the case the header is not set) by setting a query parameter 'd' This parameter is only used as a fallback. If you were hosting this on the domain example.com, you would set the fallback like so: 

<img src="https://hit.sknk.ws/c.png?d=example.com" />
